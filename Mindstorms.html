The Article

The first thing we will talk about is programing the mindstorm. When doing this you have a couple of options. You can one use the chrome book and the Lego Eve app. Two you can use an ipad with the controller app.

To code the lego eve you use the script called blocka. To code this website I have used java, html, and css.

Within the lego eve there is a main motor which controlls the drive gear or "moves forward and backwards" and their is another motor for the arm. So we will now demonstrate this.

The lego eve itself has alot more potential. The lego eve has a lazer sensor. For example if you were to put tape down on the floor in a square the lego eve would use it's sensor to follow that tape. Now of cource you cannot do this with the chromebook apps but you can do it with the windows ten version.

to get the windows ten vershion of the controller all you have to do is go to their website and download and install the program. The program works for windows ten and higher.
